# 🚀 Make My Career  
### AI-Powered Career Roadmap Platform for CSE Students

---

## 📌 Problem Statement

93% of Indian CSE students know fewer than 7 tech career options, yet no free platform offers real job market data, skill-specific roadmaps, and unbiased AI guidance tailored to them.

---

## 💡 Solution

**Make My Career** is an AI-powered platform that helps CSE students:
- Explore real career options
- View actual job market data
- Get **personalized, week-by-week roadmaps**
- Interact with an **AI mentor (text + voice)**  
- Track their learning progress

---

## ✨ Key Features

### 🔍 Google-style Career Search
Search roles, skills, companies, or cities across **12,428 real CSE job listings**

---

### 🧭 8 CSE Career Domains
- Software Development  
- Data Science & ML  
- Cloud & DevOps  
- Web Development  
- Cybersecurity  
- Data Engineering  
- Product & Tech Management  
- UI/UX & Design  

---

### 🧠 AI Skill-Specific Roadmap
- Powered by **Gemini API**
- Reads real job skills
- Generates **4-phase, week-by-week roadmap**

---

### 📅 Phase + Week Breakdown
- Weekly topics  
- Time commitment  
- Free learning resources  

---

### 🤖 AI Mentor (Voice + Text)
- Ask doubts anytime  
- Powered by Gemini API  
- Voice input using Web Speech API  

---

### 📊 User Dashboard
- Track enrolled roadmaps  
- Monitor weekly progress  
- Completion percentage  

---

### 🔐 Secure Authentication
- Email + password  
- Google Sign-In supported  

---

### 🔎 Open Data Transparency
- Dataset source publicly stated  
- No hidden ranking algorithms  

---

## 🏗️ Tech Stack

| Layer | Technology |
|------|-----------|
| AI / LLM | Google Gemini 2.0 Flash API |
| Authentication | Google OAuth 2.0 |
| Voice Input | Web Speech API (Chrome) |
| Frontend | React.js + Vite |
| UI & Fonts | Google Sans / Roboto |
| Data | Indian Job Market Dataset (12,428 CSE jobs) |
| Storage | Browser localStorage |
| Deployment | Vercel |

---

## 🔄 How It Works

1. User visits platform  
2. Searches or selects a career domain  
3. Views real job listings  
4. Applies filters (level, company, city)  
5. Opens job → sees “Why this role?”  
6. Clicks **Get Roadmap**  
7. AI reads job skills + context  
8. Generates personalized roadmap  
9. User enrolls or asks AI mentor  
10. Tracks progress on dashboard
